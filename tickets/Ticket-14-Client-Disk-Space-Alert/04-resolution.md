# Resolution – Disk Space Cleanup

## Resolution Steps
1. Logged into WIN11-01 as the affected user.
2. Identified and deleted large non-essential files created for testing.
3. Emptied the Recycle Bin to reclaim disk space.
4. Removed temporary files using:
   - Settings → System → Storage → Temporary files
5. Verified available disk space increased on the system drive.

## Security Consideration
If large or unexpected files appear suspicious, they should **not be immediately deleted**. Such files should be reviewed and, if necessary, safely analyzed according to security procedures before removal.
